# HTML / CSS Review Heuristics

- Detect global style leakage and unintended cascade effects.
- Check selector specificity battles causing fragile overrides.
- Validate responsive behavior across breakpoints.
- Ensure interactive states exist and are visible (focus/hover/disabled).
- Check theme compatibility (dark/light/high contrast tokens).
- Flag basic accessibility issues: semantic structure, contrast, focus flow.
- Watch for layout shifts that could break key user flows.
